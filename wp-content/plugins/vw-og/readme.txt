This plugin lets you edit the description, image, links and other default settings used by Open Graph for social sharing.

== Description ==

This plugin allows you to edit the default Open Graph meta tags for social sharing.

Main meta tags already added for Open Graph:

*   og:locale
*   og:url
*   og:title
*   og:description
*   og:type
*   og:image 
*   og:site_name 

Main meta tags added for Facebook Social Sharing:

*   fb:admins 
*   fb:app_id 

Main meta tags added for Twitter Social Sharing:

*   twitter:account_id
*   twitter:url 
*   twitter:card 
*   twitter:creator 
*   twitter:site 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `vw-og` to the `/wp-content/plugins/` directory;
1. Activate the plugin through the 'Plugins' menu in WordPress;
1. Edit the meta values under Configuration > VW Open Graph (You must have admin privileges to do that).
